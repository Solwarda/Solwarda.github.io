欢迎来到我的个人博客！

这是使用 Hexo + GitHub Pages 搭建的博客系统。

## 博客特点

- 简洁美观的主题
- 部署简单
- 支持 Markdown 写作

## 快速开始

1. 修改 `_config.yml` 中的站点信息
2. 在 `source/_posts` 目录下创建新的 Markdown 文件
3. 运行 `hexo server` 本地预览
4. 运行 `hexo deploy` 部署到 GitHub Pages

祝写作愉快！
